# RayShell

RayShell is a customizable shell built using [Raylib](https://github.com/raysan5/raylib).

## 🚀 Features
- Lightweight shell interface
- Built with Raylib (graphics + input handling)
- Customizable UI and behavior
- Designed for experimentation with C and low-level systems

## 📦 Requirements
- C compiler (GCC / Clang)
- Raylib installed

## 🛠️ Installation

```bash
git clone https://github.com/yourusername/rayshell.git
cd rayshell
make
```

## 🧠 Why This Exists

- This project is focused on learning:

- Building shells in C
- Handling input/output manually
- Creating UI with Raylib instead of terminal-only shells
